 Q8 : 
            "Guess the Output

function FindSum(a, b){
    return a + b;
}

function DisplayData(data, batch){
    console.log(`i am from ${data} and My batch is EA${batch}`)
}

DisplayData(""PrepBytes"", FindSum(10, 9));"

Ans : The FindSum(10, 9) function call evaluates to 10 + 9, which is 19. In the DisplayData function call, "PrepBytes" is the data parameter and 19 is the batch parameter. The template literal ${data} is replaced by "PrepBytes" and ${batch} is replaced by 19, resulting in the output: "i am from 19 and My batch is EAPrepBytes".
